# Multi-Agent AI System using Google ADK

## Overview

This project demonstrates a Multi-Agent AI system built using Google Agent Development Kit (ADK).

## Features

- Research Agent
- Summarizer Agent
- Planner Agent
- Google ADK
- Groq LLM Integration

## Tech Stack

- Python
- Google ADK
- LiteLLM
- Groq
- Prompt Engineering

## Installation

```bash
git clone <repository-url>

cd MultiAgentSystem

python -m venv .venv

.\.venv\Scripts\activate

pip install -r requirements.txt
```

## Run

```bash
adk web
```
